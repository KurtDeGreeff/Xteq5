

If this folder is located at "%PROGRAMDATA%\Xteq5\" (e.g. C:\ProgramData\Xteq5\), 
any file that was installed by SETUP.EXE will be overwritten **without notice**. 

This also includes all files starting with '-' (minus). 

Therefore, please do not change any existing file and **do not**
name your own files beginning with '-' (minus).

Files that you add will not be touched.
